# プロジェクト概要

## プロジェクト名
給与計算検定入力ソフト

## 目的
給与計算検定や各種データ入力作業のためのGUIアプリケーション。画像（PDF/写真）を表示しながら、データを効率的に入力できるツール。

## 主な機能
1. **画像表示**: PDF/画像ファイルの表示、回転、拡大縮小
2. **データ入力**: 複数のLineEditを使った効率的なデータ入力
3. **バリデーション**: リアルタイムでデータの妥当性チェック
4. **モード切り替え**: 8種類のモード（給与検定、ウタネ回覧、工場見学、外国人、防災、集英社、防災V2、工場）
5. **CSV入出力**: データの読み込み・保存（CP932エンコーディング対応）
6. **照合機能**: 入力データと既存データの照合
